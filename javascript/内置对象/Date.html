# Date

@(JavaScript)

# 概述

ECMAScript 中的 Date 类型是在早期 Java 中的 java.util.Date 类基础上构建的。为此，Date 类型使用自 UTC（Coordinated Universal Time，国际协调时间）1970 年 1 月 1 日午夜（零时）开始经过的毫秒数来保存日期。在使用这种数据存储格式的条件下，Date 类型保存的日期能够精确到 1970 年 1 月 1 日之前或之后的 285 616 年。

Date是JavaScript的一个内置函数。

console.log(typeof Date);//"function"
1

# Date()

Date对象可以作为普通函数直接调用，返回一个代表当前时间的字符串。

var date = Date();
console.log(date);
//Sun Feb 06 2022 16:05:52 GMT+0800 (中国标准时间)
console.log(typeof date);//"string"

# new Date()

1、不传入参数，返回的是当前时间。

var date1 = new Date();
console.log(date1);//Sun Feb 06 2022 15:59:46 GMT+0800 (中国标准时间)
console.log(typeof date1);//"object"

2、new Date()的参数

//参数为一组数值，一个逗号分隔
var date1 = new Date(2022, 2, 6, 0, 0, 0, 0); //Sun Mar 06 2022 00:00:00 GMT+0800 (中国标准时间)
var date1 = new Date(2022, 2, 6, 1, 2, 3, 4); //Sun Mar 06 2022 01:02:03 GMT+0800 (中国标准时间)
var date1 = new Date(2022.2, 6, 12, 12, 12, 12); //Tue Jul 12 2022 12:12:12 GMT+0800 (中国标准时间)

//参数为字符串
var date1 = new Date("2022.2.6"); //Sun Feb 06 2022 00:00:00 GMT+0800 (中国标准时间)
var date1 = new Date("2022,2,6"); //Sun Feb 06 2022 00:00:00 GMT+0800 (中国标准时间)
var date1 = new Date("2022/2/6"); //Sun Feb 06 2022 00:00:00 GMT+0800 (中国标准时间)
var date1 = new Date("2022-2-6"); //Sun Feb 06 2022 00:00:00 GMT+0800 (中国标准时间)
var date1 = new Date("2022-2.6"); //Sun Feb 06 2022 00:00:00 GMT+0800 (中国标准时间)
var date1 = new Date("2022.2-6"); //Sun Feb 06 2022 00:00:00 GMT+0800 (中国标准时间)

# 日期的运算

var date2 = new Date();
var res = date2 - date1;
console.log(res); //61041270
console.log(typeof res); //"number"

var res2 = date2 + date1;
console.log(res2); //"Sun Feb 06 2022 16:58:16 GMT+0800 (中国标准时间)Sun Feb 06 2022 00:00:00 GMT+0800 (中国标准时间)"
console.log(typeof res2); //"string"

# 实例方法

var date1 = new Date();
console.log(date1);//Mon Feb 07 2022 18:52:20 GMT+0800 (中国标准时间)

console.log(date1.toString());//Mon Feb 07 2022 18:52:20 GMT+0800 (中国标准时间)
console.log(date1.toDateString());//Mon Feb 07 2022
console.log(date1.toTimeString());//18:52:20 GMT+0800 (中国标准时间)

console.log(date1.toLocaleString());//2022/2/7 下午6:52:20
console.log(date1.toLocaleDateString());//2022/2/7
console.log(date1.toLocaleTimeString());//下午6:52:20
